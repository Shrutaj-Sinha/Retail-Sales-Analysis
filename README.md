# Retail Store Sales Analysis

## Overview
This project analyzes retail sales data to identify trends, top products, and city performance.

## Tools Used
- Excel
- SQL
- Python (Pandas, Matplotlib)

## Key Insights
- Electronics generate highest revenue
- Mumbai has highest sales
- Laptop is top revenue product

## How to Run
1. Install Python
2. Run: python analysis.py
